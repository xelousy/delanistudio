## Setup.
To get a local copy of the site:

`git clone https://github.com/xelousy/delanistudio.git`

`Extract` the cloned file.

`cd` to delanistudio folder.

*Launch* `index.html` file.

## Technologies used.
* HTML5
* CSS3
* Bootstrap
* Javascript
* JQuery
